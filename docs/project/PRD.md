# KilimCraft — Ürün Gereksinim Dokümanı
## Özet
KilimCraft, Türkiye'deki kilim ve el dokuma atölyeleri için atölye, tezgah, desen, iplik, sipariş, üretim, kalite ve sevkiyat operasyonlarını yöneten SaaS platformudur.
## Tasarım
- Renkler: indigo (#1E3A5F), saffron (#D4A017), terracotta (#C4622D), wool cream (#F7F3ED)
- Tipografi: EB Garamond + Outfit
- Düzen: icon-rail sidebar (geometric)
## Modüller
Atölyeler, Dokumacılar, Tezgahlar, Desenler, İplik Partileri, Dokuma Siparişleri, Üretim Kayıtları, Kalite Kontrolleri, Müşteriler, Sevkiyatlar
